# 🌐 Web Scraping Project

This project was developed as part of my internship task to demonstrate web scraping and data collection skills using Python.

The project extracts data from public websites and stores it in a structured dataset for analysis.

---

## 📌 Project Objective

- Extract data from websites using Python
- Understand HTML structure and web navigation
- Create custom datasets for analysis
- Store extracted data into CSV format

---

## 🚀 Technologies Used

- Python
- BeautifulSoup
- Requests
- Pandas

---

## 📂 Project Features

✅ Extracts website data automatically  
✅ Parses HTML content  
✅ Collects relevant datasets  
✅ Saves data into CSV format  
✅ Beginner-friendly implementation  

---
